# Todo App - React

Bu proje, **React** kullanılarak geliştirilmiş modern bir **Todo uygulamasıdır**.  

## Özellikler

- **Component tabanlı yapı:** Uygulama, fonksiyonel ve tekrar kullanılabilir React componentleri ile tasarlandı.  

- **Görev ekleme ve silme:** Kullanıcılar yeni görev ekleyebilir veya tamamlanan görevleri silebilir.  

- **İleri düzey React kullanımı:**  
  - **State yönetimi** ile dinamik liste güncellemeleri  
  - **Context API** veya prop aracılığıyla veri paylaşımı  
  - **Event handling** ve form validasyonları  

- **Responsive tasarım:** Farklı ekran boyutlarına uygun tasarım
